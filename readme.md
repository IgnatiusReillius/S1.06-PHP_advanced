## Sprint 1 Topic 6 - PHP advanced

This topic is focused on PHP advanced topics, such as superglobal variables, magic methods...

## Exercises

- Level 1
    - Creation of a HTML form.
    - Create a program with a couple of magic constants.
    - Overwrite one of the magic methods.

## Prerequisites

Before cloning the project, ensure the following tools are installed:

```
```
